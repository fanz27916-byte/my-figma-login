## 🎨 设计特点
基于Figma设计 (https://www.figma.com/design/l7VpDTCCgXYiGz0RolOtLN/Login-%7C-Web-Login-%7C-ziontutorial.com--Community-?node-id=1-22)：

1. **设计规范：**
   - 背景色：`#eef4f5`
   - 卡片背景：白色
   - 按钮颜色：`#0366ff` (蓝色)
   - 字体：Inter (标题) 和 IBM Plex Serif (正文)

2. **主要组件：**
   - 邮箱输入框（带图标，蓝色边框）
   - 密码输入框（带显示/隐藏切换）
   - 登录按钮（蓝色渐变）
   - 社交登录选项（3个按钮）
   - 分隔线和文本

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发服务器
```bash
npm run dev
```
访问 http://localhost:5173

### 构建生产版本
```bash
npm run build
```

## 📦 技术栈
